## Getting Started

To get a local copy up and running follow these steps.

1. Clone the repo
   ```sh
   git clone https://github.com/fabianskutta/personal-site.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run dev environment
   ```sh
   npm run dev
   ```
